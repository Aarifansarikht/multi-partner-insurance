# Multi-Partner Insurance Onboarding

A single-page insurance onboarding journey that rebrands per partner and walks a
user from browsing plans through KYC, payment and confirmation.

> Work in progress — full documentation lands with the final milestone.

## Tech stack

- React + TypeScript (Vite)
- Tailwind CSS with a semantic, token-driven theme (shadcn/ui primitives)
- Redux Toolkit for state, React Router for routing
- react-hook-form + zod for forms and validation

## Getting started

```bash
npm install
npm run dev
```
